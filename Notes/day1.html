1.What is Web Development?

Definition

Web development is the process of creating,building and maintaning websites and web application that runs on the internet.

 It involves everything from designing page layouts to handling user requests and storing data.

Two Main Branches of Web Development

Type Description
Frontend Deals with everything the user sees and interacts with in the browser.
Backend Manages the server-side, databases, and the logic that powers the website behind the scenes.
