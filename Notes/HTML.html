HTML - HYPERTEXT MARKUP LANGUAGE

Introduction to HTML:

HTML (HyperText Markup Language) is the standard language for creating web pages. It is a markup language that uses tags to structure content on a webpage, telling browsers how to display text, images, and other media.

HTML is the foundation of the web and is often used alongside CSS (for styling) and JavaScript (for interactivity).

Who invented HTML and why?

HTML was invented by Tim Berners-Lee in 1991. He created it as a way for researchers at CERN to share and structure documents online, making it easy to link and access information over the internet.

HTML is the backbone of the World Wide Web, enabling the display of text, images, and other content in browsers.

Nesting & Indentation:

HTML allows nesting, which means placing elements inside other elements. This creates a logical structure. The spacing of lines to reflect this hierarchy is called indentation, and it helps maintain clean and readable code.
