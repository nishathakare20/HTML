Text Header alighnment

1] By default, table headers are centered.

Syntax:


        <style>
            th{
                text-align:(default)center;
            }
        </style>

Example:


        <table>
            <tr>
                <th>Name</th>
                <th>city</th>
            </tr>
            <tr>
                <th>Gaytri</th>
                <th>Pune</th>
            </tr>
        </table>

Output:

Name City
Gaytri Pune

2] Text-align:Left

● use text-align: left;to Left-align content:

Syntax:


        <style>
            th{
                text-align:left;
            }
        </style>

Output

Name City
Gaytri Pune

3] Text-align:Right

● use text-align: right;to Right-align content:

Syntax:


        <style>
            th{
                text-align:right;
            }
        </style>

Output

Name City
Gaytri Pune
