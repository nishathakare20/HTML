HTML Table - rowspan Attribute

Purpose:

The rowspan attributes in HTML is used in table headers (<th>) or data cells (<td>) to make a cell span across multiple rows. It merges cells vertically in a column.

Syntax:


            <td rowspan= "number">Cell Content</td>

Example:

This table demonstrates how rowspan="2"merges the first two rows in the first column:
Name Subject Marks
math 95
Nisha science 89

Output Preview:

In the first column, the cell with "Name" spans 2 rows vertically.


Notes:

● rowspan is used to vartically merge table cells across rows.

● The number must indicate how many rows you want the cell to span.

● It's especially useful in report-style tables, attendance sheets, scorecards, etc.

● You can combine rowspan with colspan for advanced table layout.
