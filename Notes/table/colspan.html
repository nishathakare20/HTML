HTML Table colspan Attribute

Purpose:

● The colspan attribute in the HTML is used in table headers (<th>) or table data cell (<td>) to make a cell span across multiple columns.It merges the specified number of adjacement columns into one cell.

Syntax:


            <td colspan="number"> Cell Content<td>

Example:

This table show how colspan="2" merges the first two columns of the first row:
Name Age Gender City
Sarika Shinde 19 Female Nashik

Output Preview:

● The first row's "Name" cell stretches over 2 columns (merging column 1 and 2).

Notes:

● Use colspan to horizontally merge cells in a table row.

● The number you provide to colspan must match the number of columns to span.

● Helps in designing custom headers or layout cells across multiple columns.

● You can also use rowspan to merge cells vertically (across rows).
