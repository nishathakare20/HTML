HTML Table - colspan & rowspan combined

Purpose:

● Combined colspan and rowspan helps create complex table layout by merging cells both horizontally andvertically. This is useful in dashboards, report, invoices, and structured data representation.

Syntax:


            <td colspan="number"> Merged column<td>
            <td rowspan="number"> Merged Row<td>

Example:

This table cmbine colspanin the header row and rowspan in one of the data cells.
Student Info Mark
Name Subject
Amit math 90
science 85
Riya Math 88
science 92

Output Preview:

● The first header row spans two columns for "Student Info".
● The "Marks" header spans two rows.
● "Amit" and "Riya" each span two rows under the Name column.

Notes:

● colspan is used to merge cells horizontally -> across columns.
● rowspan is used to meare cells vertically -> across rows.
● Both can be used together in the same table layout.
● Ensure your total columns and rows are balanced for layout consistency.
● Useful in attendance sheets, mark reports, timetables, invoices etc.
