HTML Table

1] HTML Table Basic

        ● An HTML table is used to organize and display data in rows and columns, similar to a spreadsheet.

        ● HTML tables are created using the <table> element.

        ● A table is structured using:

            1]<tr> - Table row

            2]<th> - Table Header Cell (bold & centered by default)

            3]<td> - Table Data Cell (normal text, left-aligned)

Syntax:


            <table>
                <tr>
                   <th>table header</th>
                   <td>table data</td>
                </tr>
            </table>

2] Table Row <tr>

        ● Each row is defined using the <tr> tag.

        ● Contains <th> or <td> elements.

Example:

    
    <tr>
        <td> Name </td>
        <td> class </td>
    </tr>

Output:

Name class

3] Table Header <th>

        ● Used to define header cells.

        ● Bold and centered by default.

Example:

    
    <tr>
        <th> Name </th>
        <th> class </th>
        <th> Marks </th>
    </tr>

Output:

Name class Marks

4] Table Data Cell

    ● Defines regular data cells inside rows.

    ● Not bold, left-aligned by default

Example:

    
    <tr>
        <th> Name </th>
        <th> class </th>
    </tr>
    <tr>
        <td> Nisha </td>    
        <td> java </td>
    </tr>

Output:

Name class
Nisha Java

6] Collapsed Borders:

● Use border-collapse: collapse; to collapse borders:

Syntax:


        <style>
            table, th, td 
            {
                border: 1px solid black;
                border-collapse: collapse;
            }
        </style>

7] Cell Spacing

● to Generate a space inside a cell.

syntax: 


        <table cellspacing="5" >

8] Cell Padding

● space inside a cell.

syntax: 


        <table cellpadding="7" >

9] Tr Hover

● The tr:hover selector in CSS is used to define the style of a table row (<tr>) when the user hovers over it with the mouse pointer.


Syntax:

            
            tr:hover{
                background-color:#f2f2f2;
            }

10] Align Header Content:

● The text-align property is used to set the horizontal alignment of a text.
● A text can be centered, left or right aligned.


Syntax:


            <style>
                th{
                    text-align: center;
                }
                th{
                    text-align: left;
                }
                th{
                    text-align: right;
                }
            </style>

11]Table Border Spacing:

● Border-spacing controls the space between table cells (i.e., between <td> and <th> elements). It's particularly useful when designing non-collapsed border tables.

syntax:


        table {
            border-spacing: 10px; /* adds 10px space between cells */
        }

12] Colspan:

● Used to merge multiple columns into a single cell horizontally.

Syntax:


        <th colspan="number">Content</th>

13] Rowspan:

● Used to merge multiple Row into a single cell vertically.

Syntax:


        <td rowspan="number">Content</td>

14] Colspan-Rowspan:

● In HTML, the colspan and rowspan attributes are used within table <td> (table data) and <th> (table header) tags to merge cells horizontally and vertically, respectively. This allows for more complex table layouts and the ability to represent data that spans multiple rows or columns.

Syntax:


            <td colspan="2">merged column <td>
            <td rowspan="2">Merged row <td>

16] Responsive Table

● Tables with many columns can overflow the screen on mobile devices, causing layout issues or hidden content.

Syntax:

Wrap the <table> inside a <div> with overflow-x: auto; like this:


    <div style="overflow-x:auto;">
        <table>
            <!-- table content -->
        </table>
     </div>

📌 BONUS: Extra Table Elements in HTML

These tags enhance structure, accessibility, and styling for large or complex tables.

✅ 1. <thead> — Table Header Section

🔹 Purpose: Groups header row(s) for better structure and accessibility.

<table>
  <thead>
    <tr><th>Name</th><th>Age</th></tr>
  </thead>
</table>

✅ 2. <tbody> — Table Body Section

🔹 Purpose: Wraps the main content rows.

<table>
  <tbody>
    <tr><td>John</td><td>25</td></tr>
  </tbody>
</table>

✅ 3. <tfoot> — Table Footer Section

🔹 Purpose: Defines footer row(s) for totals or notes.

<table>
  <tfoot>
    <tr><td>Total</td><td>2 Employees</td></tr>
  </tfoot>
</table>

✅ 4. <colgroup> — Group Columns

🔹 Purpose: Group columns and apply shared styles.

<colgroup>
  <col span="1" style="background:#f2f2f2">
  <col span="1" style="background:#fff">
</colgroup>

✅ 5. <col> — Style Individual Columns

🔹 Purpose: Works within <colgroup> to target specific columns.
