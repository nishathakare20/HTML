Internal CSS

Purpose : 

            1. Define : An internal CSS is used to defined style for a single HTML pages.
                               this CSS is define in the <head> section of an html pages .
            2. For Internal CSS , We Use <style> Element In <head> Section.
            3. Selector : It is a pattern of elements and other terms that tell the browser which HTML elements should be selected
                                   to have the CSS property values inside the rule applied to them.
            4. Advantages Of Internal CSS :
                    i] Localized Styling:Keep styles within the html file,avoiding confilicts with other pages
                         and making it easier to manage style at the local level.
                    ii] Higher Specificity:Internal CSS has higher specificity than external CSS ,
                         allowing easier overriding of external styles within the same HTML file.
                    iii]Performance:Reduces HTTP request ,potentially enhancing performance as no additional CSS files need to be loaded.
                    iv]Ease of Implementation:Simple to use and implement ,making it easy to quickly apply style to a single page.
                
            5. Disadvantages of Internal CSS:
                    i]Repetition:Styles must be repeated in multiple HTML files if the same styles are needed across differnt pages.
                    ii]Increased file Size:Embedding CSS in HTML Increases the file size of the HTML document.
                    iii]Reduced Reusability:Limited code reusability code reusability across diferrent web pages.
                    iv]Limited Management:Managing style across multiple pages can become difficult without a centralized stylesheet.
