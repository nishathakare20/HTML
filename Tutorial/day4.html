Attribute in HTML

1] Only Attribute:



< input required >

2]Attribute with [Value]:

Email-ID :



< input type="email" >
< input type="password" >

3] Attribute with [property_Name & Value]:

Example : Attribute = " PropertyName : Value; "

Syntax : <p style="background-color: aqua;" >

4] Attribute with [Multiple Properties & Values]:

Example :
Attribute = " Property1 : Value1; Property2 :Value1 Value2 Value3; "

Syntax :
<p style="background-color:aqua; color:crimson; border: 5px inset blue;" >
